# Agent Project (RAG System)

这是一个基于 Python 的检索增强生成 (RAG) 系统项目。项目利用向量数据库和语言模型，实现对本地文档的智能问答与检索。

## 项目结构

* `embedding.py`: 处理文本向量化逻辑。
* `retriever.py`: 负责从向量数据库中检索相关上下文。
* `loaderDocument.py`: 负责加载和预处理不同格式的文档（如 PDF）。
* `chroma_db/`: 用于存储持久化向量数据的数据库目录。
* `prompt.py` & `FewShotPromptTemplate.py`: 提示词工程与模板管理。
* `outputParser.py`: 解析模型输出结果。

## 安装指南

### 1. 克隆项目
确保你已经获取了项目源代码并进入项目根目录：
```bash
cd agent
```

### 2. 创建并激活虚拟环境 (推荐)
使用虚拟环境可以避免依赖冲突：

* **Windows**:
    ```bash
    python -m venv venv
    .\venv\Scripts\activate
    ```
* **Linux/macOS**:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

### 3. 安装依赖
通过 `pip` 安装项目所需的库：
```bash
pip install -r requirements.txt
```

## 快速开始
目前项目包含多个功能模块，你可以根据需要运行相应的脚本，例如：
* 运行文档加载：`python loaderDocument.py`
* 运行检索测试：`python retriever.py`
